#  Iris Dataset Analysis

An exploratory data analysis (EDA) project on the classic Iris dataset using Python and Google Colab.

##  Overview
The Iris dataset contains 150 samples of iris flowers across 3 species. This project explores the data through statistics and visualizations to understand patterns and differences between species.

##  What's Inside
- Data loading & exploration
- Statistical summary
- Data visualization (distributions, scatter plots, pair plots)
- Species comparison analysis

##  Technologies
`Python` `Pandas` `Matplotlib` `Seaborn` `Google Colab`

##  Dataset
The Iris dataset contains:
- 150 samples
- 3 species: Setosa, Versicolor, Virginica
- 4 features: sepal length, sepal width, petal length, petal width
